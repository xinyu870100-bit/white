# 白飯俱樂部 GitHub Pages 更新版

這份版本是依照使用者需求重新整理的多分頁網站。

## 本版重點
- 入口頁文字「入內，讓情感開始交織」本身就能進入關於頁
- 每頁都可直接替換背景圖
- 關於、顧客守則保留單頁展示，均支援背景圖
- 店員介紹：右側 Coming soon、每張圖可點擊放大、下方可寫店員名
- 服務內容：保留圖文版型
- 菜單：左長圖、右小圖；全部可點擊放大
- 斗內香檳王：中間單張主視覺圖
- 店內環境：橫 3 直 2 排列
