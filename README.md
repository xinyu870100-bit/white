這份版本是根據原本 Canva 站的分頁順序與你後續提出的需求，重新整理的 GitHub Pages 版：

- 一頁一主題，不走長頁滾動
- 每頁右下角固定下一頁按鈕
- 底部導航 active 狀態正確
- about 改成左右分欄
- rules 置中且不爆版
- staff 改成接近原站的格狀展示

使用方式：
1. 把這個資料夾裡的所有檔案上傳到 GitHub repo 根目錄
2. 用你的圖片覆蓋 assets 內對應檔名
3. Commit 後等待 GitHub Pages 自動更新
