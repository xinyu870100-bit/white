# Rice Club 多分頁 GitHub Pages 範本

這份範本是依照原 Canva 網站的截圖重做成 **多分頁、桌機版不需捲動** 的 GitHub Pages 靜態網站。

## 目前頁面
- index.html 入口
- about.html 關於
- rules.html 顧客守則
- staff.html 店員介紹
- services.html 服務內容
- menu.html 菜單
- gallery.html 精彩瞬間
- ranking.html 斗內香檳王
- environment.html 店內環境

## 使用方式
1. 把整個資料夾上傳到 GitHub repository 根目錄。
2. 到 GitHub repository 的 **Settings > Pages**。
3. Source 選 **Deploy from a branch**。
4. Branch 選 **main**，資料夾選 **/(root)**。
5. 儲存後等待數十秒即可看到網站。

## 版面說明
- 桌機版：預設一頁一畫面、不顯示捲軸。
- 小螢幕：自動改成可捲動，避免內容被裁切。
- 底部導覽列：每頁固定一致，方便互相跳轉。
